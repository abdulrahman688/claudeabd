## Image Guidelines

### Photography Style
- Natural lighting preferred
- Warm tones (#D4AF37 gold, cream, beige)
- Authentic, not staged
- Show real women, real work
- Respectful, dignified

### Technical Specs
- Format: WebP (with JPG fallback) for web
- Compression: Optimize for web (TinyPNG, ImageOptim)
- Max file size: 200KB per image
- Resolution: 2x for retina displays

### Copyright
- Only use images with permission
- Women's photos require written consent
- Credit photographers if required

## Temporary Placeholders

While gathering real images, you can use:
- https://placeholder.com for temporary images
- https://unsplash.com for royalty-free photos (search: soap, Damascus, artisan)
- Generate simple colored rectangles with brand colors

## Adding Images

1. Place images in correct folders
2. Name exactly as listed above
3. Optimize before uploading
4. Test all pages to ensure images load
5. Check mobile performance
